# Lista de Filmes - Elite Dev Verzel

## Descrição
Aplicação full stack para pesquisa, exibição e favoritação de filmes, integrando a API do TMDb. Permite autenticação de usuário e gerenciamento de lista de favoritos.

## Tecnologias Utilizadas
- **Front-end:** React.js
- **Back-end:** Node.js + Express (ou Python Flask/Django)
- **Banco:** (Mock/Disk/json opcional ou MongoDB)
- **API de Filmes:** [The Movie Database (TMDb)](https://www.themoviedb.org/documentation/api)

## Funcionalidades
- Busca de filmes por título
- Exibição de dados do filme (imagem, título, nota TMDb)
- Favoritar/desfavoritar filmes (necessário login)
- Listagem dos filmes favoritos do usuário
